Publicado em 25 de mai. de 2020

Premier épisode d'une série sur le machine learning, branche fondamentale de l'intelligence artificielle. Dans cet épisode nous verrons les bases du machine learning avec les différents principes utilisés ainsi que les bases fondamentales des réseaux de neurones.

Prérequis : Aucun

Le machine learning est une stratégie d'apprentissage utilisée pour permettre la création de programmes capable de prévoir des situations à partir de données sans connaître la relation entre ces données et la situation.

--------------------

Les animations ont été réalisées à l'aide de manim, une librairie python open-source développée par Grant Sanderson, propriétaire de la chaîne 3blue1brown.

Minicurso de Introdução à Machine Learning e Inteligência Artificial, composto por 13 aulas, é uma iniciativa gratuita e voluntária entre pessoas.
Todos nós doamos nosso tempo para criar estes conteúdos e compartilhar contigo de forma gratuita.

Nesta aula de Bot, com Angelo Belchior, foram abordados os tópicos:

- NLP com LUIS
- Criando nosso bot com o Microsoft Bot Framework
- Testando nosso bot com o Bot Emulator
- Hospedando nosso bot no Azure Bot Service e se conectando ao Skype e Telegram
- Criando um bot a partir de uma FAQ com o Microsoft Q&A Maker

This is part 1 of the tutorial where I collect the K-Pop data and clean the dataset.

Data cleaning is an important step as you want the cleanest data for EDA and model building. If you put garbage in, then you get garbage out from the model.

In this video, I show you where I got the dataset and I go through how to clean the data so that it's usable for the exploratory data analysis (EDA) and model building phases of the data science cycle.

The dataset contains 240 K-pop fans from all over the world and 22 survey questions.

For this dataset, I re-name the columns and go through each column to see if the categories need to be simplified and re-labeled.

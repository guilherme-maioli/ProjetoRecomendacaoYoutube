Publicado em 25 de mai. de 2020

Our solution for production-grade data platform for NI.
To achieve fast time to Market for developers we had to amp up our software Development lifecycle by adding easy to use tools for various data application use cases and users.
We've created a Simple to use generic and extendable scheduled application pipeline API as Well as extending spark-testing-base Spark testing framework to accommodate the needs of our data environment. We supplied users with one-click deployment, key application metrics, and alerts created automatically.
User metrics API in Spark apps for applicative metrics. We achieved this by creating our CI/CD stack with Jenkins, Airflow, Apache Spark, Docker & Kubernetes, as Well as AWS stack with: Glue metastore, ElasticMapReduce, CloudFormation, and for monitoring we used: AWS CloudWatch, InfluxDB, Grafana, Datadog.

Panelist: Aviem Zur - Big Data Tech Lead @ Natural Intelligence; Committer & PMC Member, Apache Beam.
Specializing in data frameworks and platforms as well as open source software. Passionate about quality engineering, open source and Magic: The Gathering.

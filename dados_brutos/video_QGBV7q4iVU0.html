Minicurso de Introdução à Machine Learning e Inteligência Artificial, composto por 13 aulas, é uma iniciativa gratuita e voluntária entre pessoas.
Todos nós doamos nosso tempo para criar estes conteúdos e compartilhar contigo de forma gratuita.

Nesta aula de Visão Computacional, com Roberta Duarte, foram abordados os tópicos:
- O que é visão computacional
- História
- Desafios
- Avanços Deep Learning
- Como a visão computacional funciona?
- Sem Deep Learning x com Deep Learning
- Redes neurais convolucionais
